# TradeIQ MVP

A static MVP for a financial trading app concept with:
- buy/sell trading UI
- portfolio overview
- AI-style TradeIQ recommendations
- social trading experience
- crypto wallet mockup

## Deploy to Vercel
1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Vercel will serve the static files automatically.

## Local preview
Open the index.html file in a browser, or use any static server.
